# 音量控制使用说明

## 启动参数

设置初始音量（0-100）：
```bash
radiko.exe --volume 80
```

默认音量为 80%。

## 运行时控制

播放时可以使用以下按键控制音量：

| 按键 | 功能 |
|------|------|
| `↑` 或 `+` | 增加音量 5% |
| `↓` 或 `-` | 减少音量 5% |
| `m` | 静音/取消静音 |
| `0-9` | 直接设置音量到 0%-90% |
| `Ctrl+C` | 停止播放并退出 |

## 音量显示

播放时会显示当前音量状态：
```
Volume:  80% [████████████████░░░░]
```

- 数字显示当前音量百分比
- 进度条直观显示音量大小
- 静音时会显示 `[MUTED]` 标记

## 技术实现

音量控制通过在音频数据流层面实时调节 PCM 样本实现，因此：
- ✅ 音量调节立即生效，无需重启
- ✅ 不影响音频质量
- ✅ CPU 开销极小

## 示例

```bash
# 以 50% 音量启动
radiko.exe --volume 50

# 播放时按 ↑ 增加音量
# 播放时按 m 静音
# 播放时按 5 设置音量为 50%
```
